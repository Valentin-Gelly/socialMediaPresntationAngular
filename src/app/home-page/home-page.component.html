Présentation de l'application

Cette application vous propose une visite virtuelle interactive en 6 étapes pour vous familiariser avec différentes applications de réseaux sociaux. Voici comment elle fonctionne :

  1. Écran d'accueil : Dès le lancement de l'application, vous découvrirez un écran d'accueil qui présente les logos des principales applications sociales : Instagram, Snapchat, WhatsApp et TikTok.
  2. Sélection d'une application : En cliquant sur l'icône de l'une de ces applications, vous accédez à un menu proposant plusieurs moyens de communication. Vous aurez la possibilité de choisir celui que vous préférez.
  3. Conversation simulée : Après avoir sélectionné un moyen de communication, vous serez dirigé vers une page qui affiche une conversation typique liée à l'application choisie.
  4. Interaction avec la conversation : Vous pourrez alors cliquer pour faire apparaître une bulle de texte avec un message d'entretien en rapport avec l'image affichée. Cette étape vous permet de simuler une conversation dans l'application sélectionnée.
  5. Étape suivante : En cliquant sur le bouton "Suivant", vous accédez à l'étape suivante qui vous propose une nouvelle interaction, accompagnée d'un bouton pour "Recommencer avec un autre réseau social" ou "Quitter" l'application.
  6. Réponse physique : Après la visite virtuelle, vous pourrez répondre à des questions sur un carnet physique, mis à votre disposition près de l'écran, pour compléter l'expérience.
